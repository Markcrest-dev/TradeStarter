# Navbar Redesign and Multi-Language Update

## Changes
1. **Navbar Scroll Animation:**
   - Implemented a "pill-shaped" floating navbar when scrolling.
   - Updated `css/style.css` to transform `.header` into a compact, centered, rounded container on scroll.
   - Added dark mode support for the new design (glow effect).

2. **Multi-Language Support:**
   - Added French (fr), German (de), and Chinese (zh) to the language selector.
   - Updated all HTML pages to include these new options.
   - Verified `js/i18n-system.js` handles language switching.
